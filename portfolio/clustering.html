Clustering

What is clustering


Clustering is the process of grouping similar data together based upon common attributes. Such as if the vast majority of people whose favorite color is pink or purple are female you can cluster anyone whose favorite color is pink or purple as female. To a machine groups of data objects can be represented by using a variety of similarity metrics. The machine then tries to determine how similar objects are based upon how close they are to eachother.

K-Means

K-Means clustering is clustering sets of points into K-Clusters. It does this by first picking K randomly generated centroids (centers of clusters). It then will go through all of the points and assign that point to whichever centroid it is closest to. After this is done it will relocate the centroid to the center of all of the points that are assigned to that centroid. It will then repeat the process of re-assigning all the points to centroids and centroids moving to the new center until either the centroids stop moving or are only moving a very small amount.

Hierarchical

Hierarchical clustering starts with assuming every point is a cluster of size one. It then goes through and clusters clusters together until there are the desired number of clusters. 
  # Note: This code is from the book Collective Intelligence.
def hcluster(rows,distance=pearson):
  distances={}
  currentclustid=-1

  # Clusters are initially just the rows
  clust=[bicluster(rows[i],id=i) for i in range(len(rows))]

  while len(clust)>1:
    lowestpair=(0,1)
    closest=distance(clust[0].vec,clust[1].vec)

    # loop through every pair looking for the smallest distance
    for i in range(len(clust)):
      for j in range(i+1,len(clust)):
        # distances is the cache of distance calculations
        if (clust[i].id,clust[j].id) not in distances: 
          distances[(clust[i].id,clust[j].id)]=distance(clust[i].vec,clust[j].vec)

        d=distances[(clust[i].id,clust[j].id)]

        if d < closest:
          closest=d
          lowestpair=(i,j)

    # calculate the average of the two clusters
    mergevec=[
    (clust[lowestpair[0]].vec[i]+clust[lowestpair[1]].vec[i])/2.0 
    for i in range(len(clust[0].vec))]

    # create the new cluster
    newcluster=bicluster(mergevec,left=clust[lowestpair[0]],
                         right=clust[lowestpair[1]],
                         distance=closest,id=currentclustid)

    # cluster ids that weren't in the original set are negative
    currentclustid-=1
    del clust[lowestpair[1]]
    del clust[lowestpair[0]]
    clust.append(newcluster)

  return clust[0]
